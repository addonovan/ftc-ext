CurrentPosition val CurrentPosition: State

The current position of the ToggleServo.
DisengagedPosition var DisengagedPosition: Double

The disengaged position of the Toggle Servo, toggleOff will set the servo to this position. This value must be between 0.0,1.0.
EngagedPosition var EngagedPosition: Double

The engaged position of the Toggle Servo, toggleOn will set the servo to this position. This value must be between 0.0,1.0.
ToggleDelay var ToggleDelay: Long

The amount of time (in milliseconds) to ensure between toggle calls go through and toggle the position, by default this is 350.
