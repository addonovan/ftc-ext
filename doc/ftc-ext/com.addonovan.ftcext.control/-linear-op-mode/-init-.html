LinearOpMode()

An OpMode that runs linearly. There is no loop() method as in a regular OpMode, which is executed at a semiregular basis. A LinearOpMode will run exactly as it sounds, linearly. Once runOpMode returns, the OpMode is finished, the method will never be called again.

This is most useful for Autonomous programs.
