isNot

fun isNot(vararg modifiers: Int, lenient: Boolean = true): FieldFinder

Filters out all fields that have the given modifiers. If lenient is true, then all of the given modifiers must be present on the field for it to be removed; otherwise, only one of the flags are required to remove the field from the list.

Parameters

modifiers - The modifiers (from java.lang.reflect.Modifier to require from the fields.

lenient - If all the modifiers are required (true) or if only one is (false) to remain in the list (by default, all flags are required).

Return
This FieldFinder, so that calls may be chained.
