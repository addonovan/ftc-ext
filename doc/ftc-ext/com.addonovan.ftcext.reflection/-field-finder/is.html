fun is(vararg modifiers: Int, strict: Boolean = true): FieldFinder

Filters out all fields that dont have the required modifiers. If strict is true, then all of the given modifiers must be present on the field for it to not be filtered out; otherwise, only one of the flags must be present in order for the field to remain.

Parameters

modifiers - The modifiers (from java.lang.reflect.Modifier) to require from the fields.

strict - If all flags are required (true) or if only one is (false) to remain in the list (by default, all flags are required).

Return
This FieldFinder, so that calls may be chained.
