moveDistance

fun moveDistance(distance: Double, power: Double): Task

Creates and registers a task with the TaskManager that wil make the motor move at the given power until the given distance has been covered.

If the OpMode is linear, then whenever this method returns, the task will have been completed.

This is not guaranteed to be accurate, as it is entirely dependent upon motor encoders!

Parameters

distance - The distance to cover (in cm).

power - The power to move the motor at.

Return
The created task. (Use Task.isFinished to determine when the task has been finished).
