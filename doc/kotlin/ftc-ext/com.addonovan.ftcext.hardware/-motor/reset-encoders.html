resetEncoders

fun resetEncoders(): Task

Creates and registers a task with the TaskManager that will reset the motor encoders.

If the OpMode is linear, then whenever this method returns, the task will have already been completed.

Return
The created task. (Use Task.isFinished to determine when the task has been finished).
