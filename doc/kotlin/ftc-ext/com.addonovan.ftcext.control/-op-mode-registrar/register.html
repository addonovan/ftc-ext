register

fun register(manager: OpModeManager): Unit

Searches for all instantiable classes which inherit from AbstractOpMode and have the Register annotation and will then verify the name of the prospective OpMode and then create the correct OpModeWrapper for the class and register it with Qualcomms OpModeManager.
